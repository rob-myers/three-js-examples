# Examples using three.js

These include examples derived from questions on <https://discourse.threejs.org/>.

## Local dev

```sh
# at repo root
nvm use
npm i
npm run dev
```

## Unbundled Code Examples

```sh
# at repo root
cd src/sans-bundler
npx http-server
# navigate to http://localhost:8080/bootstrap/
```
